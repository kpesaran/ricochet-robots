Ricochet Robots Game Instructions

Objective: Move the target robot to the target cell (the cell with the chip matching the center chip) using as few moves as possible.

Movements: Use the arrow keys to move the target robot, and click to place non-target robots on the grid to help guide it.

  • Use arrow keys to move the target robot. It continues moving until it hits a wall or another robot.
  • Click to place non-target robots on the grid. Each placement counts as a move.

Goal: Reach the target cell with minimal moves
