Ricochet Robots Game Instructions

Objective: Move the target robot (the robot with the same color as the target cell chip / the center chip) to the target cell using as few moves as possible

Movements: Use the arrow keys to move the robots, and follow the same movement rules for both target and non-target robots

  • Click on a robot (target or non-target) to select it
  • Use the arrow keys to move the selected robot. It continues moving until it hits a wall or another robot

Did you find the shortest path? After solving, you can compare your solution to the shortest path to see how efficient your moves were
